Icecave\Recoil\Channel\Stream\ReadableStreamChannel

class ReadableStreamChannel implements ReadableChannelInterface

Adapts a ReactPHP readable stream into a readable channel.

Channel read operations produce strings read from the stream.

The channel is closed when the stream 'end' event is emitted.

public read()

Read a value from this channel.

Execution of the current strand is suspended until a value is available.

If the channel is already closed, or is closed while a read operation is pending a ChannelClosedException is thrown.

Read operations must be exclusive. If concurrent reads are attempted a ChannelLockedException is thrown.

Exceptions

ChannelClosedException if the channel has been closed.
ChannelLockedException if concurrent reads are attempted.
