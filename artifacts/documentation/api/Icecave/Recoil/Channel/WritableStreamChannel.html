public write(mixed $value)

[CO-ROUTINE] Write a value to this channel.

Execution of the current strand is suspended until the value has been consumed.

If the channel is already closed, or is closed while a write operation is pending a ChannelClosedException is thrown.

Write operations must be exclusive only if the underlying stream requires exlusive writes.

Parameters

mixed $value The value to write to the channel.

Exceptions

ChannelClosedException if the channel has been closed.
ChannelLockedException if concurrent writes are unsupported.
