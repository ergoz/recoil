public mixed read()

[CO-ROUTINE] Read a value from this channel.

Execution of the current strand is suspended until a value is available.

If the channel is already closed, or is closed while a read operation is pending a ChannelClosedException is thrown.

Read operations must be exclusive. If concurrent reads are attempted a ChannelLockedException is thrown.

Return Value

mixed The value read from the channel.

Exceptions

ChannelClosedException if the channel has been closed.
ChannelLockedException if concurrent reads are attempted.

public write(mixed $value)

[CO-ROUTINE] Write a value to this channel.

Execution of the current strand is suspended until the value has been consumed.

If the channel is already closed, or is closed while a write operation is pending a ChannelClosedException is thrown.

Write operations must be exclusive. If concurrent writes are attempted a ChannelLockedException is thrown.

Parameters

mixed $value The value to write to the channel.

Exceptions

ChannelClosedException if the channel has been closed.
ChannelLockedException if concurrent writes are unsupported.
