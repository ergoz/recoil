Icecave\Coro\Coroutine\CoroutineResumer

class CoroutineResumer implements TaskResumerInterface

Public interface for resuming or cancelling a suspended co-routine.

The suspended co-routine is cancelled if this object is destructed before the co-routine is resumed.
