public enqueueTask(mixed $task)

Enqueue a task for execution.

Tasks are guaranteed to be started in the order they are enqueued.

The value given must be able to be adapted into a task using the engine's task adaptor.

Parameters

mixed $task The task to execute.

public enqueue(callable $callback, array $arguments = array())

Enqueue a callback for execution.

Callbacks are guaranteed to be executed in the order they are enqueued.

If the callable returns a value that can be adapted to a co-routine it is executed.

Parameters

callable $callback The callable to invoke.
array $arguments Arguments to pass to the callable.
