Icecave\Coro\Coro

abstract class Coro

Public facade for co-routine operations.

This class contains no implementation; it is a proxy for the facade implementation provided by whichever co-routine engine is currently being used for execution.

The interface {see Icecave\Coro\Engine\FacadeInterface} details the operations that are available; some engines may provide additional operations.
