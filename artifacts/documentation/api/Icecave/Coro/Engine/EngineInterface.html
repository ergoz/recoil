public enqueueCallback(callable $callback, array $arguments = array())

Enqueue a callback for execution.

Callbacks are guaranteed to be executed in the order they are enqueued.

If the callable returns a value that can be adapted to a co-routine it is executed.

callable $callback The callable to invoke.

public enqueueDelayedCallback(integer|float $delay, callable $callback, array $arguments = array())

Enqueue a callback for execution after a given timeout.

There is no guarantee made about the order of execution of callbacks that are scheduled for execution at the same time.

If the callback returns a value that can be adapted to a co-routine it is executed.

Parameters

integer|float $delay The number of seconds to wait before invoking the callback.
callable $callback The callable to invoke.
array $arguments Arguments to pass to the callable.
