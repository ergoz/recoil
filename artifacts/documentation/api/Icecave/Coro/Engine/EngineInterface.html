public callable enqueueTask(mixed $task)

Enqueue a task for execution.

Tasks are guaranteed to be started in the order they are enqueued.

The value given must be able to be adapted into a task using the engine's task adaptor.

Parameters

mixed $task The task to execute.

Return Value

callable A function used to cancel the enqueued task.

public callable enqueue(callable $callback, array $arguments = array())

Enqueue a callback for execution.

Callbacks are guaranteed to be executed in the order they are enqueued.

If the return value of the callback can be adapted into a task, it is executed immediately.

Parameters

callable $callback The callable to invoke.
array $arguments Arguments to pass to the callable.

Return Value

callable A function used to cancel the enqueued callback.

public callable executeAfter(number $delay, callable $callback, array $arguments = array())

Enqueue a callback for execution after a given timeout.

There is no guarantee made about the order of execution of callbacks that are scheduled for execution at the same time.

If the return value of the callback can be adapted into a task, it is executed immediately.

Parameters

number $delay The number of seconds to wait before executing.
callable $callback The callable to invoke.
array $arguments Arguments to pass to the callable.

Return Value

callable A function used to cancel the enqueued callback.
