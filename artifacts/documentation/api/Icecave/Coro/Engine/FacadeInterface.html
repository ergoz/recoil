public WaitAllResult all($task1, $task2)

Wait for all of the given tasks to complete.

The return value is an object that implements ArrayAccess which contains the results from each of the tasks in the same order as they were passed.

Results are access by numeric index; accessing a result for a task that failed will throw the exception produced by that task.

$task1
