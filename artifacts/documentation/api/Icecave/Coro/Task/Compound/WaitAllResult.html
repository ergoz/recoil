Icecave\Coro\Task\Compound\WaitAllResult

class WaitAllResult implements ArrayAccess, Countable, IteratorAggregate

An array-like object that contains results of a "wait-all" operation.

The indices correspond to the order in which the tasks were passed to Coro::all().
