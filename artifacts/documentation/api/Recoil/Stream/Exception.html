Recoil\Stream\Exception

StreamClosedException Indicates that a read or write operation was attempted on a stream that is closed or that the stream was forcefully closed while read or write operations were pending.
StreamLockedException Indicates that multiple read or write operations were attempted on a stream that does not support concurrent operations.
StreamReadException Indicates that an error occured while attempting to read from a stream.
StreamWriteException Indicates that an error occured while attempting to write to a stream.
