public mixed read()

[COROUTINE] Read a value from this channel.

The implementation MUST suspend execution of the current strand until a value is available.

If the channel is already closed, or is closed while a read operation is pending the implementation MUST throw a ChannelClosedException.

The implementation MAY require read operations to be exclusive. If concurrent reads are attempted but not supported the implementation MUST throw a ChannelLockedException.

Return Value

mixed The value read from the channel.

Exceptions

ChannelClosedException if the channel has been closed.
ChannelLockedException if concurrent reads are unsupported.

in WritableChannelInterface at line 61
public close()

[COROUTINE] Close this channel.

Closing a channel indicates that no more values will be written. Once a channel is closed future invocations of write() MUST throw a ChannelClosedException.

The implementation SHOULD NOT throw an exception if close() is called on an already-closed channel.

The implementation SHOULD support closing while a write operation is in progress, otherwise ChannelLockedException MUST be thrown.

Exceptions

ChannelLockedException if the channel can not be closed due to a pending write operation.

public write(mixed $value)

[COROUTINE] Write a value to this channel.

The implementation MUST throw an InvalidArgumentException if the type of the given value is unsupported.

The implementation MAY suspend execution of the current strand until the value is consumed or internal buffers are flushed.

If the channel is already closed, or is closed while a write operation is pending the implementation MUST throw a ChannelClosedException.

The implementation MAY require write operations to be exclusive. If concurrent writes are attempted but not supported the implementation MUST throw a ChannelLockedException.

Parameters

mixed $value The value to write to the channel.

Exceptions

ChannelClosedException if the channel has been closed.
ChannelLockedException if concurrent writes are unsupported.
InvalidArgumentException if the type of $value is unsupported.
