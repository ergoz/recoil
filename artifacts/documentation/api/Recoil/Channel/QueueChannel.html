class QueueChannel implements ReadableChannelInterface, WritableChannelInterface

An unbuffered (synchronous) loop-back data channel that allows multiple concurrent read/write operations.

public mixed read()

[COROUTINE] Read a value from this channel.

Execution of the current strand is suspended until a value is available.
